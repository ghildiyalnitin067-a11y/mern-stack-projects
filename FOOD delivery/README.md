# Food Delivery Web Application

A full-stack food delivery web application built as a **project-based learning** exercise using the MERN stack. This project helped me understand how modern full-stack applications are structured, from designing responsive user interfaces to developing REST APIs, integrating databases, implementing authentication, and handling online payments.

> **Learning Project:** Built by following and extending concepts from the GreatStack MERN Stack tutorial while implementing the project independently for practice and skill development.

---

## Tech Stack

### Frontend

* React.js
* React Router
* Context API
* Axios
* CSS3

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Authentication

* JWT (JSON Web Token)
* bcrypt

### Payment

* Stripe Payment Gateway

## Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run server
```

Admin Panel

```bash
cd admin
npm install
npm run dev
```
